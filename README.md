# iPhone's Voice Memos Clone

This project is an attempt at replicating the iPhone's Voice Memos app using SwiftUI and SwiftData. Requires iOS 18.0+ to build and iOS 26.0+ for audio transcriptions. Some features are missing such as audio editing, enhanced audio, and skip silence.
